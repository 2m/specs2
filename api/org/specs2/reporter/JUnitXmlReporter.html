org.specs2.reporter

JUnitXmlReporter

trait JUnitXmlReporter extends Reporter with DefaultSelection with DefaultSequence with DefaultExecutionStrategy with JUnitXmlExporting with ConsoleOutput

This trait execute specifications and exports them as JUnit xml files in the target/test-reports directory (by default).
