MockitoStubs

trait MockitoStubs extends MocksCreation

This trait provides functionalities to declare stub values on method calls.

Usage:<code>

mockedList.get(0) returns "one" mockedList.get(0) returns ("one", "two") mockedList.get(0) throws new Exception("unexpected") mockedList.get(0) answers ( i => "value " + i.toString )

</code>

It is also possible to chain stubs like this: <code>

mockedList.get(0) returns "one" thenReturns "two" mockedList.get(0) returns "one" thenThrows new Exception("unexpected now") </code>
